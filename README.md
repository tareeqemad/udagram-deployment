# Udagram Application

Udagram is a full stack web application that allows users to register,
log in, and create image posts.

## Technologies Used
- Frontend: Ionic, Angular
- Backend: Node.js, Express
- Database: PostgreSQL (AWS RDS)
- Cloud Services: AWS S3, Elastic Beanstalk
- CI/CD: CircleCI
